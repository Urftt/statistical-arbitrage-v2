# T01: Scaffold Next.js project with Mantine dark theme

**Scaffolded Next.js 16 + Mantine v8 dark-themed frontend with Plotly template constant ported from Dash**

## What Happened

1. Ran `create-next-app@latest` with App Router, TypeScript, ESLint, no Tailwind, no src dir.
2. Installed `@mantine/core@8.3.17`, `@mantine/hooks`, `@tabler/icons-react`.
3. Created `next.config.ts` with `optimizePackageImports` for Mantine tree-shaking.
4. Created `lib/theme.ts` with `createTheme()` (blue primary, system fonts) and `PLOTLY_DARK_TEMPLATE` — all 17 values ported exactly from the Dash `layout.py` template.
5. Replaced default `app/layout.tsx` with Mantine root layout: `@mantine/core/styles.css` import, `mantineHtmlProps` on `<html>`, `ColorSchemeScript defaultColorScheme="dark"` in head, `MantineProvider` wrapping children.
6. Deleted `globals.css` and `page.module.css`. Created minimal `app/page.tsx` with Mantine Text/Title components.
